
# Program 1
Prerequisites for running program 1:
> Should have JDK/JVM installed
> Environment path should be set.

Use the below command to execute it.

  - javac HelloWorld.java
  - java -Xmx128M -Xms16M HelloWorld 8(where 8 is n)
